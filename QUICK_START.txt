⚡ QUICK START - LANDING PAGE

🎯 ABRIR LA PÁGINA
→ Abre: public/landing.html en tu navegador

📝 PERSONALIZAR (5 minutos)
1. Busca "+57 (300) 123-4567" → Reemplaza con tu teléfono
2. Busca "info@kcafeancestral.com" → Reemplaza con tu email
3. Busca "Selva Tropical, Valle del Cauca" → Reemplaza con tu ubicación

🖼️ CAMBIAR IMÁGENES
1. Descarga/prepara tus imágenes (1600x900px mínimo)
2. Guárdalas en: public/assets/images/landing/
3. Reemplaza URLs de Unsplash en landing.html

🔗 CONECTAR BOTONES
1. Abre: public/js/landing-optimizer.js
2. Busca: LANDING_CONFIG
3. Actualiza URLs en la sección "urls"

📱 PROBAR RESPONSIVIDAD
- Chrome/Firefox DevTools (F12)
- Device Toolbar (Ctrl+Shift+M)
- Prueba en 375px, 768px, 1920px

🚀 DEPLOY
- Copia public/landing.html + css/ + js/ + assets/ a tu servidor
- Asegúrate de usar HTTPS
- Testing final en producción

📚 DOCUMENTACIÓN
- INSTALLATION_GUIDE.md (Guía completa)
- LANDING_README.md (Detalles técnicos)
- LANDING_PAGE_RESUMEN.md (Overview completo)

✅ YA INCLUIDO
✓ 9 secciones (Hero, Cabañas, Paquetes, etc)
✓ Responsive design (mobile, tablet, desktop)
✓ Menú hamburguesa mobile
✓ Animaciones suaves
✓ Dark mode
✓ Lazy loading
✓ SEO ready
✓ Accesibilidad WCAG

⚠️ IMPORTANTE
- Las URLs de imágenes son de Unsplash (prueba)
- Reemplaza con tus propias imágenes para producción
- Optimiza imágenes: mogrify -resize 1600x900 *.jpg
- Convierte a WebP: cwebp -q 80 imagen.jpg -o imagen.webp

💡 TIPS
- Usa window.LANDING en DevTools para debugging
- Los comentarios en el código explican cada parte
- Las variables de color están en css/variables.css
- JavaScript es modular y reutilizable

🆘 PROBLEMAS COMUNES
- Imágenes no cargan → Verifica rutas relativas
- Menú mobile no funciona → Asegúrate que landing.js está cargado
- Estilos no aplican → Limpia cache (Ctrl+F5)

¿PREGUNTAS?
→ Revisa los archivos .md
→ Lee los comentarios en el código
→ Abre DevTools para debugging
